Coin-Kept backend provides RESTful APIs for managing personal
financial records including income and expense transactions.

This repository contains the backend service of the Coin-Kept application.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Features
- User registration and login
- JWT-based authentication
- CRUD operations for transactions
- Transaction categorization (income / expense)
- Secure protected routes

## Project Overview
- Designed RESTful API structure
- Implemented authentication using JWT
- Built MongoDB schemas for financial records
- Applied middleware for authorization and validation
